# Plan de Estudio - Java 21

Repositorio personal para aprender Java desde cero hasta un nivel intermedio-avanzado.

## Requisitos

- Java 21 o superior
- Editor de texto o IDE (VS Code, IntelliJ IDEA, etc.)

## Cómo usar este repositorio

1. Cada módulo tendrá su propia carpeta
2. Dentro de cada carpeta habrá ejemplos y ejercicios
3. Actualizar el estado en este README al completar cada tema
